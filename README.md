# GitHub Analyzer 🔍

A simple Python project that fetches GitHub user data using APIs.

## Features
- Get user information (followers, following)
- Get total repositories
- Show top repositories with stars
- Show followers list

## Technologies Used
- Python
- Requests Library
- GitHub API

## How to Run
1. Install requests:
   pip install requests

2. Run the script:
   python github_analyzer.py

3. Enter a GitHub username

## Example
Input:
octocat

Output:
- Username
- Followers
- Repositories
- Top repos
- Followers list
